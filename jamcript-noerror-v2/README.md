# Jamcript (No-error deploy v2)

✅ Deploys on Vercel with **no env vars** and avoids build/export issues.  
✅ Includes a real root route `/` (no 404).

## Deploy (no terminal)
1) Upload all files to GitHub (make sure `app/` is at repo root)
2) Import to Vercel as a Next.js project
3) Deploy

If you still see 404, it usually means you're opening an old domain. Use the domain shown under the latest Production Deployment.
